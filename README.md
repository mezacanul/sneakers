# React Front-End Template

This custom template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

It is crafted as a lightweight boilerplate starter solution to jumpstart any SPA for any Front End project.

## It includes the following stack already configured:

-   TypeScript
-   Tailwind CSS
-   Tailwind merge function
-   Redux (with a default Slice and Provider already set)
-   React Router
-   useResponsive (custom hook for responsive values in hooks/useResponsive.ts)
